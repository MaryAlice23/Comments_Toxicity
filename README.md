## PROJECT TITLE
Deep Learning for Comment Toxicity Detection with Streamlit

-----------------------------------------------------

## PROJECT OVERVIEW

This project develops a machine learning based system that automatically detects toxic comments in online platforms using Natural Language Processing and Deep Learning techniques.

Online communities and social media platforms generate a large number of comments every day. Many of these comments contain harmful content such as insults, hate speech, harassment, and abusive language. Manually moderating such content is difficult and time consuming.

To address this problem, this project implements an automated toxicity detection system that analyzes textual comments and predicts whether they are toxic or non-toxic.

The trained machine learning model is integrated into a Streamlit web application that allows users to interact with the system easily.

-----------------------------------------------------

## FEATURES

• Real-time toxicity prediction for user comments  
• NLP based text preprocessing and feature extraction  
• Deep learning model for toxicity classification  
• Streamlit web application interface

## HOW TO RUN THE PROJECT

Step 1: Clone the repository

Step 2: Install required libraries

pip install -r requirements.txt

Step 3: Run the Streamlit application

streamlit run app.py

Step 4: Open the application in browser

http://localhost:8501

-----------------------------------------------------

## APPLICATION FUNCTIONALITIES

1. Enter a comment in the text input box
2. Click the Predict button
3. The system analyzes the comment
4. Displays whether the comment is toxic or non-toxic

Additional feature:
Users can upload CSV files to analyze multiple comments at once.
